# team-profile-generator

Team Profile Generator

Description:

This project requires inquirer@8.2.4, NodeJS, and Jest. Node needs to be installed for this project. 
index.HTML file will be generated according to the information the provided
use the terminal to open index.js and then enter the following comand "node index.js" in the terminal.
Enter information based on the questions asked.
Based on the information the user provided, index.HTML will be created in the dist folder.
